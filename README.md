TRABAJO FINAL DATA SCIENCE.
Análisis de la evolución musical entre éxitos de distintas épocas.
Este repositorio contiene la investigación cuantitativa para el análisis comparativo entre éxitos de la música entre los años 80-90 y la actualidad.
Autora: Jara López
Asignatura: Data Science/ Profesor: Modesto Escobar
Instrucciones de reproducibilidad del análisis
1. Descargar 'conjunto de datos.csv' y 'TRABAJOFINAL_MUSICA.R' en la misma carpeta.
2. Abrir el script en RStudio y pulsar 'Run'.
3. Seleccionar el archivo .csv en la ventana emergente flotante, tras hacer run en la línea 8: mis_datos <- read.csv(file.choose()) 
   ( Se incluye set.seed(123) y sample_n para cumplir la aleatoriedad de la muestra).
